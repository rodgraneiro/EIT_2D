Isotropic mesh with Hua virtual electrode (EIT)

This page presents the results of the inverse isotropic problem using Hua's virtual electrode
model in Electrical Impedance Tomography (EIT) for three different mesh configurations. Each
row shows the results for ten different lambdas.
The figures below show:

  • The conductivity distribution of the phantom;
  • The reconstructed conductivity obtained from the inverse problem;
  • The influence of different regularization parameters λ.
  • Results obtained for a maximum of 100 iterations.

Simulation Parameters

Current pattern: Skip - 3
Electrodes: 16
Domain conductivity: σxx = 3 , σxy = 0, σyy = 3 S/m and angle 0°.
Object conductivity: σxx = 3 , σxy = 0, σyy = 1 S/m and angle 0°.
Numbers of phantom elements mesh: 2038, 2660 and 2892 respectively.
Number of elements in the base mesh: 910
Maximum iterations: 100
Regularization: Gaussian HPF
Virtual electrode model: Hua
lambdas = np.logspace(-4, 1, 10)
