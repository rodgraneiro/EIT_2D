Results of the Anisotropic Inverse Problem EIT with Hua virtual electrode

Anisotropic domain with an Square object phantom - Angle 0°

This page presents the results of the anisotropic inverse problem using the Hua virtual electrode model in Electrical Impedance Tomography (EIT). The figures below show:

  • The conductivity distribution of the phantom;
  • The anisotropy;
  • The reconstructed conductivity obtained from the inverse problem;
  • The influence of different regularization parameters λ.

Object 1 conductivity: σxx = 3 , σxy = 0, σyy = 1 S/m and angle 0°.
Object 2 conductivity: σxx = 1 , σxy = 0, σyy = 3 S/m and angle 0°.

Number of elements in the base mesh: 1164

The anisotropic conductivity tensor is represented by:

$$ \sigma = \begin{bmatrix} \sigma_{xx} & \sigma_{xy} \\ \sigma_{xy} & \sigma_{yy} \end{bmatrix} $$

Results obtained for a maximum of 50 iterations.         Lambda values are obtained by: lambdas = np.logspace(-5, 0, 20).
