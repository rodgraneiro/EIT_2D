Results of the Anisotropic Inverse Problem EIT with Hua virtual electrode

Anisotropic rectangular domain - Zero Objects - Angle 0° - Auto scale

This page presents the results of the anisotropic inverse problem using the Hua virtual electrode model in Electrical Impedance Tomography (EIT).
In this test, the phantom consists of a rectangular mesh with anisotropic conductivity of [3-0-1] S/m and an angle of 0°.
The base mesh used in the solution of the inverse problem is rotated at 0, ±30, and ±60°. The results are presented below.
The figures below show:

  • The conductivity distribution of the phantom;
  • The electric field behavior caused by anisotropy;
  • The reconstructed conductivity obtained from the inverse problem;
  • The influence of different regularization parameters λ.

Simulation Parameters

Current pattern: Skip - 3
Electrodes: 16
Domain conductivity: σxx = 3 , σxy = 0, σyy = 1 S/m and angle 0°.
Number of phantom elements mesh: 2004
Number of elements in the base mesh: 710
Maximum iterations: 5
Regularization: Gaussian HPF
Virtual electrode model: Hua
lambdas = np.logspace(-9, 1, 20)
